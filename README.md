# Demo Website

A beginner-friendly responsive demo website built with HTML, CSS, and JavaScript.

This project started as a responsive navigation bar exercise and grew into a small multi-section website with dark mode, a mobile hamburger menu, smooth scrolling, and a contact form with simple validation.

## Live Demo

https://hammerheart92.github.io/demo-website/

## Features

- Responsive navigation bar
- Mobile hamburger menu
- Smooth scrolling navigation
- Home, About, Services, and Contact sections
- Services cards
- Contact form
- Basic form validation with JavaScript
- Dark / light mode toggle
- Responsive layout for desktop and mobile
- Footer

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages
